# Brand Extractor Agent

An AI-powered brand extraction agent built with the Botpress Agent Development Kit (ADK) that extracts brand colors, logos, and visual identity from any website.

## Demo

Try it live: **[https://brand-extractor.botpress.bot](https://brand-extractor.botpress.bot)**

![Brand Extractor Demo](preview.png)

## Features

- Extracts brand colors from any website URL or company name
- Automatically discovers company websites from company names
- Captures and analyzes multiple pages for accurate color extraction
- Generates light and dark theme color palettes
- Exports colors in CSS, JSON, or Tailwind format
- Built on Botpress ADK for extensibility
