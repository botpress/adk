# Deep Research Agent

An AI-powered research agent built with the Botpress Agent Development Kit (ADK) that performs comprehensive web research on any topic.

## Demo

Try it live: **[https://deep-research.botpress.bot](https://deep-research.botpress.bot)**

## Features

- Automated web research and information gathering
- Synthesizes findings into comprehensive reports
- Built on Botpress ADK for extensibility
