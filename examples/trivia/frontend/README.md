# Guardrails Frontend

A React web interface demonstrating topic guardrails with visual feedback.

See the [main README](../README.md) for full documentation.

## Quick Start

```bash
bun install
bun run dev
```

## Configuration

Update `src/config/constants.ts` with your webchat client ID after deploying the bot.

## Project Structure

```
src/
├── components/    # Message renderers & GuardrailMessage
├── config/        # Bot configuration
└── hooks/         # Custom hooks
```
